# Fake_Breaking_News_Factory  📰

A simple Python project that generates random news headlines using predefined subjects, actions, and places.

## Features
- Random headline generation
- User input handling (yes/no loop)
- Uses Python's random module

## Technologies Used
- Python

## How to Run
1. Clone the repo
2. Run the Python file:
   python Fake_Breaking_News_Factoey.py

## Example Output
BREAKING NEWS: Prime minister will be flying in car
